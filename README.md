# Saras Finance - Frontend Internship Assignment

A premium, intelligent search interface designed for finance, audit, and legal professionals. Built as a submission for the Saras Finance frontend engineering internship.

## üöÄ Features Implemented

### 1. Core Requirements
- **Intelligent Search Interface:** Responsive search bar with mock API integration.
- **Search Results Display:** Clean, card-based layout showing title, snippet, and detailed descriptions.
- **Skeleton Loader:** Premium "shimmering card" loading state with enforced visibility for perceived performance.
- **Infinite Scrolling:** Automatically loads more results when reaching the bottom of the page.
- **API Integration:** Connects to `https://postman-echo.com/post` to simulate real-world backend interaction.

### 2. "All Out" Creative Overhaul (Bonus)
- **Cinematic Experience:**
    - **Animated Mesh Background:** A living, breathing gradient background.
    - **Mouse Spotlight:** An interactive light source that follows the cursor.
    - **3D Parallax Tilt:** Search result cards tilt in 3D space based on mouse position.
    - **Glassmorphism:** Frosted glass headers with noise texture for a tangible, premium feel.

- **Advanced Interactivity:**
    - **Physics-based Animations:** Smooth height expansion using Vue `<Transition>` hooks.
    - **Keyboard Shortcuts:** `Ctrl+K` to focus, `Esc` to blur/clear.
    - **Theme Toggle:** Seamless morphing animation between Dark and Light modes.

- **Brand Integration:**
    - **Custom Brand Identity:** Integrated Saras Finance logo and brand colors.
    - **External Dashboard Link:** Functional "Go to Dashboard" button in the header.
